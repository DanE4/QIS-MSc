## Learning objectives

- Explain the structure of optimizing compilers and phase responsibilities.
- Explain relations between high-level language, IR, analysis, and optimization.
- Explain code generation for different NISQ hardware targets.
- Implement key optimization phases.
- Derive and use analytic performance models.
- Evaluate quantum computer performance.
- Explain relationships between HPC and quantum systems.
- Identify performance bottlenecks in quantum computing stacks.
