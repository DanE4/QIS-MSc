## Learning objectives

- Understand relationships in the Python scientific-computing ecosystem.
- Use git and GitHub for small open-source project workflows.
- Develop reusable, efficient, and documented scientific code.
- Analyze and simulate quantum optical circuits.
- Model and simulate open quantum-system dynamics.
- Quantitatively analyze noisy measurement data.
- Profile code and apply acceleration strategies.
- Execute a time-constrained end-to-end computational project.
- Present computational methods and results clearly.
